
# 🧺 QuiickErrand
> **The Reliable Best Friend for Nigerian Market Logistics.**

QuiickErrand is a decentralized platform designed to formalize the informal errand-running economy in Nigeria. We bridge the trust gap between busy individuals (**Requesters**) and local shoppers (**Runners**) using milestone-based escrow and community-driven price transparency.

---

## 🚀 The Vision
In many Nigerian communities, "sending someone to the market" is a daily occurrence handled via fragmented WhatsApp chats and verbal agreements. **QuiickErrand** standardizes this experience by:
* **Securing Funds:** No more "disappearing" with market money.
* **Standardizing Fees:** Transparent labor costs based on effort, not guesswork.
* **Guaranteing Quality:** A reputation system that rewards the best "pickers."

## 🛠 Core Features
* **Milestone Escrow:** Funds are locked and released in stages (Initial Float → Final Settlement).
* **Live Price Ledger:** A crowd-sourced database of real-time commodity prices in local markets (e.g., Mile 12, Mushin, Bodija).
* **Proof-of-Purchase (PoP):** Mandatory receipt and item photo uploads for transparency.
* **Dispute Mediation:** A fair system for handling quality issues or price discrepancies.

## 🏗 Technical Stack (Proposed)
* **Frontend:** Progressive Web App (PWA) using React/Next.js (Mobile-first, low-data consumption).
* **Backend:** Node.js with a PostgreSQL database for the Price Ledger.
* **Trust Layer:** Stellar Network (using smart-contract logic for low-fee escrow and instant settlement).
* **Notifications:** WhatsApp Business API integration for real-time alerts.

## 📈 Roadmap
- [ ] **Phase 1:** MVP – Manual price entry and basic Escrow flow.
- [ ] **Phase 2:** Beta – Deployment in a specific local government area (LGA).
- [ ] **Phase 3:** Scaling – Automated Price Ledger and "Pro-Runner" certification.
